# Cyber Cheatsheet ⚡

Curated collection of pentesting and cybersecurity commands for quick reference.

## Live Site

https://gnaixnaij.github.io/cyber-cheatsheet/

## Sections

- Reconnaissance (nmap, whois, dnsrecon, dig)
- Scanning & Enumeration (gobuster, ffuf, nikto, wpscan)
- Web Exploitation (sqlmap, curl)
- Exploitation Frameworks (msfconsole, msfvenom, searchsploit)
- Password Cracking (hashcat, john, hydra)
- Reverse Shells & Listeners (netcat, bash, python, socat)
- Lateral Movement (impacket, evil-winrm, smbclient)
- Cryptography & Encoding (openssl, base64, xxd)
- Misc Utilities (linpeas, winpeas, steghide, binwalk, strings)

## Features

- Clean, dark-mode UI
- Copy-to-clipboard on every command
- Keyboard-navigable table of contents
- Mobile responsive

- HTML5 · CSS3 · JavaScript (vanilla)

Browse by section, click any command to copy it. No installations needed — it's a static site.
